# Jira Automated Testing Framework

## Test Cases

- **Test**: loginToJira
- **Test**: createNewProject
- **Test**: createNewStory
- **Test**: createNewBug
- **Test**: linkBugToStory

## Tech Stack
- Java
- Selenium WebDriver
- Maven

## Running the Tests
1. Clone the repository.
2. Navigate to the project directory.
3. Run `mvn test` to execute the test suite.

## Note
The tests are annotated with order for execution.
